# FLYING BUCKET 
A simple demonstrator with coaxial motors and thrust vecoring, nicely packet into a 5" tube.
Based upon the dRhem flight controller V1.3

## Electronics
- Teensy 4.1
- MPU6050
- KISS ESC32 via OneShot
